# Changelog

## 0.1.0 (2026-03-08)

### Features

- **CDD Explorer** — TreeView sidebar with modules, skills, rules, and agents
- **Init Project** — Create CDD project with preset selection via Quick Pick
- **Add Module** — Add module with role and stack picker
- **Add/Change Stack** — Change technology stack for existing modules
- **Run Doctor** — Health check for CDD infrastructure
- **Regenerate** — Regenerate all `.claude/` infrastructure
- **CDD Agent** — AI-powered specialist chat with streaming responses
- **Anthropic Auth** — Secure API key storage via VS Code SecretStorage
- **Status Bar** — Project info and agent connection status
- **Auto-refresh** — TreeView updates on `cdd.json` changes
